おすすめの学習方法
  1. 自分に合ったレベルから始める
  2. 単語を分類しながら覚える
  3. 覚えてきたら4択テスト
  4. くり返す
サービス利用のヒント

学習状況を Google Spreadsheets に保存すると、様々な端末で学習できます。 学習状況を保存したい スプレッドシートID を入力し、Google アカウントでログインしてください。 初回ログイン時の スプレッドシートID は未入力で大丈夫です。ドキュメントを新しく作成します。
